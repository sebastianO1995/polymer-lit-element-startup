This is a start up project for lit-element webcomponents using the polymer cli.
Implements the fetch polyfill for IE 11.

Browser support:

# Chrome (latest)

# Safari

# Firefox

# IE 11

# Edge 18

Commands

# run npm install

To run the server

# polymer serve

To build
Reference: https://polymer-library.polymer-project.org/3.0/docs/tools/polymer-json

# polymer build
